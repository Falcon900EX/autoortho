Auto Ortho for Silicon Mac - Beta Package

This beta package is a portable Python launcher with local dependencies.

How to start:
1. Unzip the package.
2. Double-click:
   Start Auto Ortho for Silicon Mac.command
3. On first launch, it creates a local runtime/venv from the bundled wheelhouse.
4. The AutoOrtho GUI opens.
5. Click Run to start the FUSE-T launcher.
6. Launch X-Plane after the Terminal says the FUSE-T mounts are ready.

Requirements:
- Apple Silicon Mac recommended.
- Python 3 must be installed for the first beta package format.
- FUSE-T must be installed separately.
- X-Plane 12 should be installed.
- At least one AutoOrtho scenery region should be installed from the Scenery tab.

Included:
- AutoOrtho source files
- Silicon Mac GUI changes
- generic FUSE-T launcher
- native Apple Silicon dylibs
- local Python dependency wheelhouse

Notes:
- This beta does not install macFUSE.
- This beta uses FUSE-T.
- This beta creates temporary mounts under ~/AutoOrthoMounts.
- The GUI Run checkbox controls whether AutoOrtho stops automatically when X-Plane quits.
- If something fails, check the Terminal output and logs on the Desktop.
